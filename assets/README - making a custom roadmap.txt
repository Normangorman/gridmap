Roads:
| = vertical road
- = horizontal road

Junctions:
> = vertical junction left
< = vertical junction right
v = horizontal junction up
^ = horizontal junction down
+ = horizontal and vertical junction

{ = corner junction top left
} = corner junction top right
[ = corner junction bottom left
] = corner junction bottom right

Interactive:
G = garage (must be aligned with appropriate junction)

Background:
T = tower block
any unrecognized symbol = blank grey background
